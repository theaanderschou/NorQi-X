NORQi Styleguide

Vi er 3 studerende fra Multimediedesign på UCL, der arbejder på vores 2. semester eksamen, en eksamen vedrørende NORQi's brugerflade, hvorfor vi har udviklet denne styleguide. Denne living styleguide er en videre guide til designet på vores løsning til NORQi, og indeholder derfor både design-elementer samt code snippets.

Logo

Anvendelse: Logo skal bruges på alle relevente sider. Logoet skal placeret til venstre, øverste hjørne på forsiden.

Code Snippet (HTML):

              

<div class="logo-line">
  <a href="index.html">
    <img src="img/NORQI-Logo.svg" alt="Logo">
  </a>
</div>

Code Snippet (CSS):

  width: 100%;
  z-index: 2;
  position: absolute;
}

  width: 110px;
  margin: 20px 0 0 20px;
}

Farver

HEX #FFFFFF

HEX #313331

HEX #5CB720

Anvendelse: De tre ovenstående farver bruges igennem hele websitedesignet, og afspejler de farver NORQi bruger på deres hjemmeside. Den hvide farve (HEX #FFFFFF) bruges til baggrund samt tekst elementer som har brug for en anden kontrastfarve, eksempelvis teksten på billeder. Den mørkegrå farve (HEX #313331) bruges til al tekst på hvid baggrund, samt til at dele siden op i sektioner. Den grønne farve (HEX #5CB720) bruges til knapper, links, logo, menu og ikoner samt i footeren.

Navigation

Anvendelse: Navigationsbaren kommer til at være til højre på alle siderne, da den skal hjælpe brugeren med at navigere rundt. Navagationsbaren indeholder 6 punkter, der hver fører til en separat side. Som f.eks. vores 3D manual side, hvor man som bruger kan få en gennemgang af kipgryden og alle dens funktioner.
